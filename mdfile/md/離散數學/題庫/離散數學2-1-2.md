
> [!NOTE]+ QUESTION
> Use set Builder notation to give a description for below sets.
>  ( a ) {0,3,6,9,12}
>  ( b ) {-3,-2,-1,0,1,2,3}
>  ( c ) {m,n,o,p}

> [!NOTE]- Ans
> ( a ) => $\{x|x\mod 3 = 0\ and\ 0\le x\le 12\}$
> ( b ) => $\{x|x\in Z^+\ and\ -3 \le x\le 3\}$
> ( c ) => $\{x|x is a letter of string "mnop"\}$
