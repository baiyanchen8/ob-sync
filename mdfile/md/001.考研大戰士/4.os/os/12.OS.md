# I/O Haedware
## I/O Haedware
- Variety of I/O device
	- Storage
		- HDD、SSD
	- Transmission
		- internet interface
	- Human-interface
		- keyboard
- ***Common concept***
	- **Port**
		- connection point for device
		- ex: thunderbolt、usb port、SATA、HDMI...
	- **Bus**
		- A set of *wire* and define *protocol*
		- Can be *daisy chain* 、p2p、or hub(1-多)
			- *daisy chain*:通常是指將多個裝置依序串接在一起，形成一個鏈狀結構。
