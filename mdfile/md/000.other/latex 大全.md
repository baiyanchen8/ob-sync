### normal math

| ---------- | ------ |
| `$=$`      | $=$    |
| `$\neq$`   | $\neq$ |

|            |        |
### 希臘字母
| 發音     | latex code  | 大寫        | 小寫        |
| ------ | ----------- | --------- | --------- |
| Sigma  | `$\sigma$`  | $\Sigma$  | $\sigma$  |
| forall | `$\forall$` | $\forall$ |           |
| exists | `\exists`   | $\exists$ |           |
| lambda | `\lambda`   | $\Lambda$ | $\lambda$ |
|        |             |           |           |
